## 1. Introduction

The Zappy project is a networked game where teams of AI players compete to reach level 8 first. It consists of three independent binaries: a server that manages the game world, a GUI client for visualization, and an AI client that controls a player.

Everything passes from the server.

### Architecture

```
┌──────────┐     ┌──────────┐     ┌─────────┐ 
│    AI    │ <-- │  Server  │ <-- │   GUI   │
|          | --> |          | --> |         |
└──────────┘     └──────────┘     └─────────┘
```

The communication is made by a TCP Protocol, defined here: [Zappy Protocol](https://github.com/EpitechPGE2-2025/G-YEP-400-PAR-4-1-zappy-2/blob/feature/documentation/docs/Zappy_Protocol.md).
